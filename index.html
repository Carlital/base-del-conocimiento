HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Describe la estructura de una página web utilizando una serie de elementos y etiquetas.

CSS

CSS (Cascading Style Sheets) se usa para dar estilo a las páginas web. Permite definir el diseño, la apariencia y el formato de un documento HTML.

JavaScript

JavaScript es un lenguaje de programación que se utiliza para crear contenido dinámico e interactivo en las páginas web, como juegos, animaciones y formularios interactivos.
